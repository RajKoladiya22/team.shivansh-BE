# Project Roadmap

## Future Enhancements
- **Lead Automations**: Auto-assign leads using round-robin distribution to active sales agents.
- **Advanced Attendance Logs**: Integrate biometric check-in feeds or physical device APIs.
- **Invoice Automations**: Automatically generate invoices from quotations upon client approval.
- **Chat Enhancements**: Implement channel-based messaging, file attachments, and video conference integrations.

# AI Project Knowledge

## Key System Concepts
- **Roles**: Access tiers (`ADMIN`, `EMPLOYEE`, `SALES`, `INTERN`).
- **Learning Journey**: Skill enhancement tasks marked by `isLearning: true` flag.
- **Demos**: Indicator of customer interest for Leads, tracked by `demoCount` and status checks.
- **User Product Expertise**: Skill alignment matrix mapping developer levels (BEGINNER to EXPERT) to products.
- **DSU (Daily Standup)**: Daily status updates mapped to projects and tasks.

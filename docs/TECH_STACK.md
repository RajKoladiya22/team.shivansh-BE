# Technical Stack

## Backend Runtime & Language
- **Node.js**: Server environment runtime.
- **TypeScript**: Typed JavaScript compiler targeting ES2022.

## Frameworks & Libraries
- **Express.js (v5.0)**: HTTP routing server.
- **Prisma Client (v7.8)**: Modern, type-safe database client and query builder.
- **Socket.IO (v4.8)**: Bidirectional, low-latency WebSocket manager.
- **Zod**: Declarative schema validation for request payloads.
- **Jose / Jsonwebtoken**: JWT creation and verification.
- **Bcrypt / Bcryptjs**: Password hashing with salt factor 12.
- **Nodemailer**: Email processing via SMTP.
- **ExcelJS / XLSX**: Spreadsheet exports and parser logic.
- **Puppeteer**: PDF rendering engine for salary payslips.
